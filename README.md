# Indienstein - AI游戏灵感生成器

<div align="center">
  <h3>🎮 基于AI的游戏创意灵感生成工具</h3>
  <p>利用多维度信息空间和人工智能，帮助游戏开发者快速生成创意灵感</p>

## ✨ 功能特性

### 🎯 核心功能
- **多维度灵感生成**: 基于游戏主题、机制、美术风格、音效等多个维度生成创意
- **AI驱动**: 支持多家AI服务商（硅基流动、深度求索、智谱AI）
- **批量生成**: 支持一次性生成多个灵感，提升创作效率
- **智能配置**: 可调节AI参数（温度、最大令牌数等）
- **灵活导出**: 支持详细模式和简化模式的导出功能

### 📊 信息空间
- **多维度支持**: 游戏主题、核心机制、美术风格、目标平台等
- **丰富向量库**: 每个维度包含大量精心设计的信息向量
- **权重系统**: 支持向量权重，影响生成概率
- **概览统计**: 实时显示维度数量、向量总数等统计信息

### 💾 数据管理
- **本地存储**: API密钥和配置信息安全存储在本地
- **历史记录**: 自动保存生成历史，支持重新查看
- **导出功能**: 支持文本格式导出，便于后续整理

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 稳定的网络连接

### 使用步骤

1. **配置API密钥**
   - 打开应用后进入"总览"页面
   - 选择要使用的AI服务商
   - 输入对应的API Key并保存

2. **生成灵感**
   - 切换到"Indienstein"页面
   - 选择所需的维度（可选多个）
   - 添加自定义提示词（可选）
   - 调整生成参数
   - 点击"生成灵感"

3. **查看和导出**
   - 查看生成的灵感内容
   - 选择导出模式（详细/简化）
   - 导出为文本文件

## 🔧 配置说明

#### 硅基流动 (推荐)
- **注册地址**: https://cloud.siliconflow.cn/
- **邀请码**: `UZbmtVXt` (使用邀请码可获得2000万免费token)
- **支持模型**: DeepSeek、GLM、Qwen等

#### 深度求索 (DeepSeek)
- **注册地址**: https://www.deepseek.com/
- **特点**: 强大的推理能力
- **支持模型**: deepseek-chat、deepseek-coder等

#### 智谱AI (GLM)
- **注册地址**: https://chatglm.cn/
- **特点**: 中文理解能力强
- **支持模型**: GLM-4、GLM-4V等

### 生成参数说明

- **Temperature (0-1)**: 控制生成的随机性
  - 0.1-0.3: 更稳定、一致的输出
  - 0.7-0.9: 更有创意、多样的输出
- **Max Tokens**: 控制生成内容的最大长度
- **生成数量**: 批量生成时的数量
- **请求间隔**: 批量生成时的间隔时间

## 📖 使用指南

### 维度选择策略
1. **无选择**: 系统从所有维度随机选择，获得完全随机的灵感
2. **单维度**: 专注于特定方面的创意生成
3. **多维度**: 获得更丰富、多角度的灵感组合
4. **重复维度**: 可多次添加同一维度，增加该维度的影响权重

### 提示词优化
- 使用具体、描述性的语言
- 可以指定游戏类型、目标受众等
- 避免过于复杂的指令
- 可以要求特定的输出格式

### 批量生成建议
- 适当设置请求间隔，避免API限制
- 生成数量建议控制在10以内
- 大批量生成时关注API使用量

## 🎮 信息维度详解

### 游戏主题 (Theme)
包含科幻、奇幻、现实、恐怖等多种主题设定，为游戏提供世界观基础。

### 核心机制 (Mechanism)
涵盖回合制、实时、建造、解谜等游戏玩法机制，定义游戏的核心交互方式。

### 美术风格 (Art Style)
包括像素、写实、卡通、抽象等视觉风格，影响游戏的整体美术表现。

### 目标平台 (Platform)
支持PC、移动端、主机等不同平台，影响游戏的设计和技术选择。

### 音效风格 (Audio)
包含环境音、背景音乐、音效设计等音频相关的创意元素。
